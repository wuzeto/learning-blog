# 数字电子技术复习笔记
## 一、计算相关核心公式
1. **香农采样定理**：采样频率需大于模拟信号最高频率的2倍，即$f_s > 2f_{max}$，以避免信号混叠。
2. **存储器存储容量计算**：存储容量 = $2^{\text{地址线数}} \times \text{位线数}$（单位：bit），例如13根地址线、8位位线的存储器容量为$2^{13} \times 8 = 8192 \times 8 = 65536$ bit（即64KB）。
3. **D/A转换输出电压计算**：$V_o = -\frac{V_{REF}}{2^n} \times D$，其中$V_{REF}$为参考电压，$n$为输入数据位数，$D$为输入数字量（十进制值，如4位数字量$d_3d_2d_1d_0$对应$D=8d_3+4d_2+2d_1+d_0$）。
4. **A/D转换关键参数计算**：
   - 分辨率：$\text{分辨率} = \frac{1}{2^n - 1}$（无单位，$n$为输入数据位数，分辨率越小精度越高）；
   - 量化阶距：$\Delta U = \frac{V_{max}}{2^n}$（$V_{max}$为输入模拟信号最大值）；
   - 最大量化误差：$\text{最大量化误差} = \frac{\Delta U}{2} = \frac{V_{max}}{2^{n+1}}$；
   - 转换时间上限：$T_{max} = \frac{1}{2f}$（$f$为采样频率）。
5. **555定时器电路参数计算**：
   - 单稳态触发器暂稳态时间：$T_W = 1.1RC$（$R$为定时电阻，$C$为定时电容）；
   - 多谐振荡器振荡周期：$T = (R_1 + 2R_2)C\ln2$（充电回路$R_1+R_2$，放电回路$R_2$）；
   - 施密特触发器阈值电压：上限阈值$V_{T+} = \frac{2}{3}V_{CC}$，下限阈值$V_{T-} = \frac{1}{3}V_{CC}$，回差电压$\Delta V_T = V_{T+} - V_{T-} = \frac{1}{3}V_{CC}$。


## 二、核心概念背诵
### 2.1 集成电路基础
1. 双极型集成电路的典型电路是**TTL电路**，单极型集成电路的典型电路是**CMOS电路**；按制造工艺，集成电路分为**单极型**和**双极型**，其中**单极型（CMOS）** 抗干扰能力更强。
2. TTL门电路标准参数：输出高电平约$\boxed{3.5V}$，输出低电平约$\boxed{0.2V}$，阈值电压约$\boxed{1.4V}$；TTL输出低电平时带**灌电流负载**，负载电流增大时输出低电平会升高。
3. 数字电路中，晶体三极管主要工作在**饱和区**和**截止区**（放大区仅用于模拟电路）。


### 2.2 逻辑函数与门电路
1. 逻辑函数的四种表示方法：**真值表**、**卡诺图**、**逻辑表达式**、**逻辑图**。
2. 特殊功能门电路：
   - OC门（集电极开路门）：需外接电源和上拉电阻，可实现**线与逻辑**，还可用于电平转换、驱动负载（如发光二极管）；
   - TSL门（三态门）：有高电平、低电平、高阻态三种输出，使能端控制状态（低电平有效或高电平有效）；
   - TG门（传输门）：双向传输，$C=1、\overline{C}=0$时导通，$C=0、\overline{C}=1$时截止，可传输模拟信号。
3. 可编程逻辑器件（PLD）结构差异：
   | 器件类型 | 与阵列 | 或阵列 | 核心特点 |
   |----------|--------|--------|----------|
   | PROM     | 固定   | 可编程 | 仅或阵列可编程，用于存储固定数据 |
   | FPLA     | 可编程 | 可编程 | 与、或阵列均可编程，灵活性高 |
   | PAL      | 可编程 | 固定   | 或阵列固定，结构简单 |
   | GAL      | 可编程 | 固定   | 含可编程输出逻辑宏单元（OLMC），可重复编程 |
4. GAL器件：全称**通用阵列逻辑**，采用**电可擦CMOS（E²CMOS）** 工艺，通过编程**结构控制字**设定**输出逻辑宏单元（OLMC）** 工作模式，支持重复编程。


### 2.3 触发器与时序电路
1. 触发器分类：
   - 按动作特点：**基本型**、**同步型**、**主从型**、**边沿型**（边沿型仅在CP上升/下降沿触发，抗干扰能力最强）；
   - 按逻辑功能：**RS触发器（RSF）**、**JK触发器（JKF）**、**T触发器（TF）**、**T'触发器（T'F）**、**D触发器（DF）**，特性方程如下：
     - D触发器：$Q^{n+1} = D$；
     - JK触发器：$Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n$；
     - T触发器：$Q^{n+1} = T\oplus Q^n$（$T=0$保持，$T=1$翻转）；
     - T'触发器：$Q^{n+1} = \overline{Q^n}$（每来一个CP翻转一次，计数功能）。
2. 数字系统分类：按组成方式分为**组合逻辑电路**（无记忆，输出仅取决于当前输入）和**时序逻辑电路**（有记忆，输出取决于当前输入和历史状态）。
3. 组合逻辑电路竞争冒险：内因是**传输延时**（不同路径信号到达时间不同，导致瞬时错误输出）。


### 2.4 常用集成电路功能
1. 555定时器应用电路：
   - **施密特触发器**（2个稳定状态）和**单稳态触发器**（1个稳定状态）用于脉冲整形；
   - **多谐振荡器**（0个稳定状态）用于产生方波脉冲；
   - 宽脉冲转窄脉冲需用**积分单稳态触发器**（施密特触发器无法控制脉冲宽度）。
2. A/D转换器：
   - **双积分型ADC**：抗干扰能力强，转换速度慢；
   - **逐次逼近型ADC**：转换速度快，抗干扰能力弱。
3. 半导体存储器：结构包含**地址译码器**、**存储矩阵**、**输出缓冲器**；RAM扩展分为**字扩展**（增加存储单元数量，扩展地址线）和**位扩展**（增加每个单元位数，扩展位线）。
4. 典型集成芯片参数：
   - EPROM2864：13个地址输入端，8个数据输出端；
   - 74LS290（十进制计数器）：由2进制和5进制计数器组成，$CP_0$接输入时$Q_0$为2进制，$CP_1$接$Q_0$时构成10进制；
   - 74LS161（4位二进制计数器）：同步置数、异步清零，最大计数值15（1111）；74LS160（十进制计数器）：同步置数、异步清零，最大计数值9（1001）。


## 三、逻辑代数与逻辑函数化简
### 3.1 基本运算与定律
1. 三种基本逻辑运算：与（$\cdot$）、或（$+$）、非（$\overline{\quad}$）；五种复合运算：与非、或非、与或非、异或（$\oplus$）、同或（$\odot$）。
   - 异或运算：$A\oplus B = \overline{A}B + A\overline{B}$（两变量不同为1）；
   - 同或运算：$A\odot B = AB + \overline{A}\overline{B}$（两变量相同为1），且$\overline{A\odot B} = A\oplus B$（同或取反为异或）。
2. 核心定律（需熟练应用）：
   | 定律类型 | 公式1 | 公式2 |
   |----------|--------|--------|
   | 分配律 | $A(B+C) = AB + AC$ | $A + BC = (A+B)(A+C)$ |
   | 反演律（德摩根） | $\overline{AB} = \overline{A} + \overline{B}$ | $\overline{A+B} = \overline{A}\overline{B}$ |
   | 吸收律 | $A(A+B) = A$ | $A + AB = A$ |
   | 其他 | $A + \overline{A}B = A + B$ | $(A+B)(\overline{A}+C)(B+C) = (A+B)(\overline{A}+C)$ |

3. 重要规则：
   - 对偶定理：逻辑式$Y$中“$\cdot$”与“$+$”、0与1互换，得对偶式$Y'$，若$Y=Z$则$Y'=Z'$；
   - 反演定理：逻辑式$Y$中“$\cdot$”与“$+$”、0与1互换，原变量与反变量互换，得反函数$\overline{Y}$，需遵守“先括号、再与、后或”的运算顺序，非单个变量的反号保留不变（如$\overline{Y} = \overline{A(B+C) + CD} = (\overline{A} + \overline{B}\overline{C})(\overline{C} + \overline{D})$）。


### 3.2 逻辑函数化简方法
#### （1）代数法化简
通过上述定律消去多余的与项和变量，例如：
$$
\begin{align*}
L &= AD + A\overline{D} + AB + \overline{A}C + BD + A\overline{B}EF + \overline{B}EF \\
&= A(1) + AB + \overline{A}C + BD + \overline{B}EF \quad (\text{吸收律：}AD + A\overline{D} = A) \\
&= A + \overline{A}C + BD + \overline{B}EF \quad (\text{吸收律：}A + AB = A) \\
&= A + C + BD + \overline{B}EF \quad (\text{吸收律：}A + \overline{A}C = A + C)
\end{align*}
$$

#### （2）卡诺图法化简
- 适用场景：变量数$\leq 4$的逻辑函数，直观且不易出错。
- 核心原则：
  1. 卡诺图单元格对应最小项，相邻单元格（包括上下左右、四角）最小项仅1位变量不同；
  2. 合并最小项时，圈的大小必须为$2^n$（$n=0,1,2,3$，即1、2、4、8个单元格）；
  3. 圈的数量越少越好，覆盖所有“1”（圈1法）或“0”（圈0法，求反函数后再取反），每个圈至少包含1个未被其他圈覆盖的单元格。

- 示例（圈0法）：  
  若卡诺图中“0”对应的反函数$\overline{L} = B\overline{C}\overline{D}$，则原函数$L = \overline{B\overline{C}\overline{D}} = \overline{B} + C + D$。

#### （3）含无关项的化简
无关项（约束项/任意项）是不可能出现的输入组合，卡诺图中用“×”表示，可灵活视为“0”或“1”以简化表达式。例如：红绿灯电路中，“红、绿、黄同时亮”为无关项，化简时可利用该无关项合并更多最小项。


### 3.3 逻辑函数的门电路实现
以“最简与或式转与非-与非式”为例（用与非门实现）：
1. 将逻辑函数化为最简与或式；
2. 对最简与或式二次取反（$\overline{\overline{L}} = L$）；
3. 利用反演律将内层取反转化为与非运算，例如$L = AB + CD = \overline{\overline{AB + CD}} = \overline{\overline{AB} \cdot \overline{CD}}$；
4. 根据与非-与非式画逻辑图（仅用与非门）。


## 四、组合逻辑电路分析与设计
### 4.1 核心集成芯片功能
1. **74LS138（3线-8线译码器）**：
   - 输入3位地址$A_2A_1A_0$，输出8位低电平有效信号$\overline{Y_0} \sim \overline{Y_7}$；
   - 使能端$G_1=1、\overline{G_2A}=\overline{G_2B}=0$时工作，可用于多输出组合逻辑电路设计（如实现多个逻辑函数的与非表达式）。

2. **74LS283（4位超前进位全加器）**：
   - 输入：4位被加数$A_3A_2A_1A_0$、4位加数$B_3B_2B_1B_0$、初始进位$C_0$；
   - 输出：4位和$S_3S_2S_1S_0$、最终进位$C_4$；
   - 应用：加法运算、代码转换（如BCD码转余三码，加3即可）。

3. **74LS151（8选1数据选择器）**：
   - 输入3位地址$A_2A_1A_0$、8路数据$D_0 \sim D_7$、低电平有效使能端$\overline{S}$；
   - 输出$Y = \sum_{i=0}^7 m_iD_i$（$m_i$为地址最小项），可实现单输出组合逻辑函数。

4. **74LS153（双4选1数据选择器）**：
   - 含2个独立4选1模块，共用地址$A_1A_0$，各自有使能端$\overline{S_1}、\overline{S_2}$；
   - 应用：双输出组合逻辑电路设计。


### 4.2 组合逻辑电路分析步骤
1. 写出输出端的逻辑表达式（从输入到输出，逐级推导）；
2. 化简逻辑表达式（代数法或卡诺图法）；
3. 列出真值表（输入变量所有组合对应的输出值）；
4. 分析逻辑功能（如奇偶校验器、不一致电路、全加器/全减器等）。

#### 示例：不一致电路分析
- 逻辑表达式：$L = \overline{ABC}(A+B+C) = \overline{ABC + \overline{A+B+C}} = \overline{ABC + \overline{A}\overline{B}\overline{C}}$；
- 真值表：当$A、B、C$全为0或全为1时，$L=0$；否则$L=1$；
- 功能：判断三个变量是否“不一致”，不一致时输出1。


### 4.3 组合逻辑电路设计步骤
1. 抽象问题：定义输入/输出变量，确定0/1对应的实际含义（如红绿灯电路中，$A=1$表示红灯亮，$L=1$表示汽车通行）；
2. 列真值表：根据功能需求，列出所有输入组合对应的输出值；
3. 写逻辑表达式：将真值表中输出为1的最小项相加，得到最小项表达式；
4. 化简表达式：根据器件类型（SSI门电路、MSI芯片、PLD）选择最简形式；
5. 画逻辑图：根据化简后的表达式，选用对应器件绘制电路。

#### 示例：用74LS151实现三变量多数表决电路
- 功能：3个输入中多数为1时输出1，表达式$L = AB\overline{C} + A\overline{B}C + \overline{A}BC + ABC$；
- 设计：令$A_2=A、A_1=B、A_0=C$，则$D_0=0、D_1=0、D_2=0、D_3=1、D_4=0、D_5=1、D_6=1、D_7=1$，使能端$\overline{S}=0$；
- 输出：$Y = L$，实现多数表决功能。


## 五、时序逻辑电路分析与设计
### 5.1 时序电路基本概念
1. 描述方法：**状态转换表**、**状态转换图**、**时序图**（波形图）；
2. 分析步骤：
   1. 写出驱动方程（触发器输入信号的表达式）和输出方程；
   2. 写出状态方程（将驱动方程代入触发器特性方程）；
   3. 列状态转换表（计算每个现态对应的次态和输出）；
   4. 画状态转换图和时序图；
   5. 分析功能（如计数器、寄存器、序列信号发生器等）。


### 5.2 常用时序集成芯片应用
#### （1）74LS194（4位双向移位寄存器）
- 控制端功能：
  - $\overline{R_D}$：异步清零（低电平有效）；
  - $S_1S_0$：工作模式（00保持、01右移、10左移、11并行置数）；
  - $D_{IR}$（右移输入）、$D_{IL}$（左移输入）：移位数据输入；
- 应用：环形移位寄存器（$Q_3$接$D_{IR}$，实现模4计数）、扭环形移位寄存器（$Q_3$接$\overline{D_{IR}}$，实现模8计数）。

#### （2）计数器设计（以74LS161为例）
74LS161为4位二进制同步计数器，可通过“异步清零法”或“同步置数法”设计N进制计数器：
1. **异步清零法**（适用于$N \leq 16$）：
   - 原理：当计数器计到$N$时，产生清零信号（接$\overline{R_D}$），立即清零；
   - 示例：设计10进制计数器，计到10（1010）时，$\overline{R_D} = \overline{Q_3Q_1}$，状态为0000~1001（10个状态）。

2. **同步置数法**（适用于任意$N$）：
   - 原理：当计数器计到“目标态”时，产生置数信号（接$\overline{LD}$），下一个CP到来时置数为“初始态”；
   - 示例：设计10进制计数器，初始态0000，目标态1001（9），当计到9时，$\overline{LD} = \overline{C}$（进位输出$C=1$），下一个CP置数为0000，状态为0000~1001（10个状态）。

3. **多芯片级联**（适用于$N > 16$）：
   - 同步级联：前级进位输出$C$接后级$EP、ET$，所有芯片CP同步；
   - 异步级联：前级某输出（如$Q_3$）接后级CP，异步触发；
   - 示例：两片74LS160（十进制）级联29进制计数器，整体置数法中，当计到28（00101000）时，产生置数信号，下一个CP置数为00000000。


### 5.3 序列信号发生器与顺序脉冲发生器
1. **序列信号发生器**（74LS161+74LS151）：
   - 74LS161作为地址发生器，输出3位地址$A_2A_1A_0$；
   - 74LS151作为数据选择器，根据序列需求设置$D_0 \sim D_7$；
   - 示例：产生序列11101000，令$D_0=1、D_1=1、D_2=1、D_3=0、D_4=1、D_5=0、D_6=0、D_7=0$，CP触发后输出对应序列。

2. **顺序脉冲发生器**（74LS161+74LS138）：
   - 74LS161产生3位地址$A_2A_1A_0$（000~111）；
   - 74LS138将地址译为8路低电平脉冲$\overline{Y_0} \sim \overline{Y_7}$；
   - 功能：CP触发时，依次输出8路脉冲，实现“顺序控制”。


## 六、半导体存储器与可编程逻辑器件
### 6.1 半导体存储器
1. **RAM（随机存取存储器）**：
   - 结构：地址译码器（行译码+列译码）、存储矩阵（存储单元阵列）、输出缓冲器；
   - 扩展：字扩展（增加地址线，如2片1K×8RAM扩展为2K×8）、位扩展（增加位线，如2片1K×4RAM扩展为1K×8）。

2. **ROM（只读存储器）**：
   - 类型：PROM（一次性编程）、EPROM（紫外线擦除）、EEPROM（电擦除）；
   - 应用：存储固定数据（如查表法实现组合逻辑函数），例如用ROM实现BCD码转七段显示码。


### 6.2 可编程逻辑器件（PLD）
1. **PLA（可编程逻辑阵列）**：与阵列和或阵列均可编程，直接用最简与或式编程，节省芯片资源。
2. **GAL（通用阵列逻辑）**：
   - 核心：可编程输出逻辑宏单元（OLMC），支持5种工作模式（专用输入、专用组合输出、带反馈组合输出、时序组合输出、寄存器输出）；
   - 优势：电可擦除、重复编程、加密功能、电子标签，灵活性高于PAL。


## 七、数模与模数转换
### 7.1 D/A转换器（DAC）
1. **权电阻网络DAC**：
   - 结构：权电阻网络（电阻值$2^nR$，$n$为位权）、电子开关、运算放大器；
   - 输出电压：$V_o = -\frac{V_{REF}}{2^n} \sum_{i=0}^{n-1} 2^i d_i$（$d_i$为第$i$位数字量，0或1）。
2. **性能指标**：分辨率（$\frac{1}{2^n - 1}$）、转换精度（实际输出与理论值的误差）、转换速度（建立时间）。


### 7.2 A/D转换器（ADC）
1. **转换过程**：采样（按香农定理）→ 保持（稳定采样值）→ 量化（将模拟量转为离散值）→ 编码（转为数字量）。
2. **逐次逼近型ADC**：
   - 原理：通过比较器逐位确定数字量的每一位（从最高位到最低位）；
   - 特点：转换速度快（$\leq 1\mu s$），精度中等，适用于高速场景。
3. **双积分型ADC**：
   - 原理：两次积分（对输入电压积分→对基准电压反向积分），转换结果与输入电压平均值成正比；
   - 特点：抗干扰能力强（抑制工频干扰），转换速度慢（$\geq 10ms$），适用于高精度、低速场景（如数字万用表）。


## 八、典型例题解析
### 例题1：74LS283构成加减运算电路
已知74LS283为4位全加器，$A$为控制端，$X、Y$为4位输入，分析功能：
1. 当$A=0$时，$B_i = Y_i$，电路为$X + Y$，$Z = X + Y$，$W = C_4$（进位）；
2. 当$A=1$时，$B_i = \overline{Y_i}$（异或1取反），电路为$X + \overline{Y} + 1 = X - Y$（补码减法），$W = C_4 \oplus A$（符号位，$W=0$表示$X \geq Y$，$W=1$表示$X < Y$）；
3. 功能：$A=0$加法，$A=1$减法，$W$为减法符号位。

### 例题2：555定时器单稳态电路设计
要求输出1s的正脉冲，$R=27k\Omega$，求$C$：
- 公式：$T_W = 1.1RC$；
- 计算：$C = \frac{T_W}{1.1R} = \frac{1}{1.1 \times 27 \times 10^3} \approx 33.7\mu F$，取标称值33μF。

### 例题3：74LS161设计10进制计数器
- 异步清零法：当计到10（1010）时，$\overline{R_D} = \overline{Q_3Q_1}$，状态为0000~1001（10个状态）；
- 同步置数法：初始态0000，当计到9（1001）时，$C=1$，$\overline{LD} = \overline{C}$，下一个CP置数为0000，状态为0000~1001。

# 电子技术基础知识点整理

## 第一章 半导体器件基础

### 二极管
- **三大特性**：
  - 热敏性：温度影响载流子浓度和迁移率
  - 光敏性：光照产生电子-空穴对
  - 掺杂性：通过掺杂控制导电类型
- **PN结形成**：
  - N型半导体：多子为电子（5价元素掺杂）
  - P型半导体：多子为空穴（3价元素掺杂）
  - 扩散运动（浓度差驱动）+漂移运动（内电场驱动）=动态平衡
- **温度影响**：
  - 温度↑ → 导通电压U<sub>D(on)</sub>↓
  - 温度影响少数载流子浓度
- **掺杂决定**：
  - 掺杂类型决定多数载流子
  - 温度决定少数载流子浓度
- **基本特性**：
  - 电中性条件
  - 耗尽层（空间电荷区）

### 三极管（BJT）
- **结构特点**：
  - 两种载流子参与导电（电子和空穴）
  - 流控电流源特性
  - 发射区：高掺杂
  - 基区：很薄
  - 集电结：面积大
- **工作机理**：
  - 扩散运动 → I<sub>E</sub>
  - 复合 → I<sub>B</sub>
  - 漂移 → I<sub>C</sub>
- **工作状态**：
  - **临界饱和**：U<sub>CE</sub>=U<sub>BE</sub>，U<sub>CES</sub>=0.7V/0.3V（硅/锗）
  - **反反截止**：发射结反偏，集电结反偏
  - **正正饱和**：发射结正偏，集电结正偏
- **状态判断**：
  - **放大区**：0＜I<sub>B</sub>＜I<sub>BS</sub>
  - **饱和区**：I<sub>B</sub>＞I<sub>BS</sub>
  - **截止区**：I<sub>B</sub>=0
- **温度影响**：
  - 温度↑ → β↑，I<sub>B</sub>和I<sub>C</sub>↑
  - 输出特性曲线：间隔增大且上移
  - U<sub>BE</sub>↓
- **三种组态**：
  | 组态 | 电压放大倍数 | 输入电阻 | 输出电阻 | 特点 |
  |------|--------------|----------|----------|------|
  | 共集 | A<sub>u</sub>≈1 | 最大 | 最小 | 射极跟随器，电压跟随 |
  | 共射 | 大（负值） | 中等 | 中等 | 常用放大电路 |
  | 共基 | 大（正值） | 最小 | 最大 | 频带最宽 |
- **耦合方式**：
  - **阻容耦合**：
    - 优点：工作点独立、信号损失小、温漂小
    - 缺点：无法集成、低频特性差
  - **直接耦合**：
    - 缺点：静态工作点不独立、温漂严重
- **频率响应**：
  - A<sub>u</sub> = A<sub>um</sub>/(1+jf/f<sub>H</sub>)(1+f<sub>L</sub>/jf)
  - **截止失真**：Q点偏低
  - **饱和失真**：Q点偏高

### 场效应管（FET）
- **特点**：
  - 只有一种载流子参与导电
  - 电压控制器件
- **分类**：
  - 无U<sub>GS</sub>时：增强型（无沟道）、耗尽型（有沟道）
- **工作区域**：
  - 可变电阻区
  - 恒流区
  - 夹断区
- **转移特性**：
  - I<sub>DQ</sub> = I<sub>DSS</sub>(1 - U<sub>GS</sub>/U<sub>GS(off)</sub>)<sup>2</sup>

### 集成电路
- **组成结构**：
  1. **输入级**：差分放大电路
     - A<sub>c</sub>小、A<sub>d</sub>大、R<sub>i</sub>大
  2. **中间级**：共射放大电路
     - 主要电压放大，常采用复合管
  3. **输出级**：OCL甲乙类互补对称功率放大电路
     - R<sub>o</sub>小，带负载能力强
  4. **偏置电路**：电流源电路
- **耦合方式**：直接耦合、阻容耦合、变压器耦合、光电耦合

### 电流源电路
- **作用**：
  - 提供稳定的静态工作点电流
  - 作为有源负载替代大电阻，提高放大倍数
- **分类**：
  - 偏置用电流源：提供Q点
  - 有源负载：取代大电阻
- **典型电路**：
  - 镜像电流源
  - 比例电流源
  - 微电流源

### 差分放大电路
- **输入信号**：
  - 差模输入：u<sub>1d</sub> - u<sub>2d</sub>
  - 共模输入：(u<sub>1d</sub> + u<sub>2d</sub>)/2
- **性能指标**：
  - 差模放大倍数
  - 共模抑制比K<sub>CMR</sub>

### 功率放大电路
- **特点**：
  - 工作在极限状态
  - 非线性失真小
- **工作状态对比**：
  | 类型 | 导通角 | 失真 | 效率 | 管耗 |
  |------|--------|------|------|------|
  | 甲类 | 2π | 最小 | 最低（≤50%） | 最大 |
  | 乙类 | π | 最大 | 最高（≤78.5%） | 最小 |
  | 甲乙类 | (π,2π) | 中等 | 中等 | 中等 |
- **选管原则**：
  - P<sub>CM</sub> > 0.2P<sub>om</sub>
  - U<sub>(BR)CEO</sub> > 2V<sub>CC</sub>
  - I<sub>CM</sub> > V<sub>CC</sub>/R<sub>L</sub>
- **交越失真**：乙类功放特有

### 反馈
- **分类**：
  - 非线性应用：开环/正反馈
  - 线性应用：负反馈
- **反馈组态判断**：
  - 电压/电流反馈
  - 串联/并联反馈
  - 正/负反馈

### 集成运放
- **理想特性**：
  - A<sub>ud</sub> = ∞
  - R<sub>i</sub> = ∞
  - K<sub>CMR</sub> = ∞
  - R<sub>o</sub> = 0
- **基本运算电路**：
  - 反相比例运算
  - 同相比例运算
  - **电压跟随器**（特殊同相比例）
  - 差分运算（减法运算）
  - 积分运算：u<sub>o</sub> = -1/RC∫u<sub>i</sub>dt + u<sub>o</sub>(t<sub>1</sub>)

### 振荡电路
- **RC桥式振荡电路**：
  - RC串并联选频网络 + 正反馈
  - 稳幅元件：R<sub>f</sub>负温度系数，R<sub>1</sub>正温度系数
  - 二极管稳幅电路
- **振荡四环节**：
  1. 放大电路
  2. 选频网络
  3. 正反馈网络
  4. 稳幅环节
- **LC振荡电路**：
  - 变压器反馈式：L<sub>1</sub>+L<sub>2</sub>+2M
  - 电容三点式：C<sub>1</sub>//C<sub>2</sub>
- **振荡条件**：
  - 幅度条件：|AF| = 1
  - 相位条件：φ<sub>A</sub> + φ<sub>F</sub> = 2nπ

### 比较器
- **单限比较器**：一个阈值
- **滞回比较器**：两个阈值，抗干扰能力强
- **窗口比较器**：两个阈值，判断是否在区间内

### 直流稳压电源
- **组成环节**：
  1. **变压器**：提供合适的交流电压
  2. **整流电路**：将交流变为单向脉动电压
  3. **滤波电路**：平滑脉动电压
  4. **稳压电路**：输出稳定的直流电压
- **整流电路对比**：
  | 类型 | 输出直流电压 | 二极管承受最大反向电压 | 纹波系数 |
  |------|--------------|------------------------|----------|
  | 单相半波 | 0.45U<sub>2</sub> | √2U<sub>2</sub> | 大 |
  | 单相桥式 | 0.9U<sub>2</sub> | √2U<sub>2</sub> | 较小 |
  | 桥式+电容滤波 | (0.9~1.2)U<sub>2</sub>（取1.2） | √2U<sub>2</sub> | 小 |
- **滤波条件**：RC ≥ (3~5)T/2
- **串联型稳压电路组成**：
  1. 取样电路
  2. 基准电压
  3. 比较放大电路
  4. 调整管（向负载提供大电流）
- **三端稳压器**：
  - CW7805：输出5V，1.5A
  - CW78M05：输出5V，0.5A
  - CW78L05：输出5V，0.1A

### 频率响应分析
- **中频段**：耦合电容短路，极间电容开路
- **低频段**：考虑耦合电容，极间电容开路
- **高频段**：考虑极间电容，耦合电容短路

### 稳压管参数计算
- **限流电阻选择**：
  - R<sub>min</sub> = (U<sub>Imax</sub> - U<sub>Z</sub>)/(I<sub>Zmax</sub> + I<sub>Lmin</sub>)
  - R<sub>max</sub> = (U<sub>Imin</sub> - U<sub>Z</sub>)/(I<sub>Zmin</sub> + I<sub>Lmax</sub>)
